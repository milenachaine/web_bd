Ce site permet d'effectuer des recherches sur une base de données contenant des liens vers des ressources informatiques (tutoriels, logiciels, etc.). Depuis cette page, il est possible de chercher une ressource au hasard et d'effectuer une recherche textuelle sur l'ensemble de la base. Depuis l'onglet 'Recherche', il est possible de remplir un formulaire pour effectuer une recherche spécifique. Enfin, depuis l'onglet 'Ajout', il est possible de suggérer des ajouts à la base.
